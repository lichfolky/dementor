# Dementor

Broadcast a local midi stream converting it in a websocket data stream

--- Nodes

## Typescript for node

npm i -D typescript

## Websocket

[websocket](https://github.com/websockets/ws)

```
npm install ws
```

## Midi to node

[node-midi](https://github.com/justinlatimer/node-midi)

```
npm install midi
```
